My Personal Blog App
====================

Overview:
---------
This is a simple personal blog web application where you can write your daily thoughts, save them, and view previous entries. It works completely offline using your browser’s localStorage—no backend or database required.

Files Included:
---------------
1. index.html     - The main webpage layout
2. style.css      - Styling and design using modern CSS
3. script.js      - JavaScript logic to handle saving and displaying blog posts
4. README.txt     - This documentation

Features:
---------
- Add new blog entries with a title and content
- Automatically saves posts in your browser's local storage
- Responsive, clean UI design
- View all previous entries in reverse chronological order
- Completely offline-capable

How to Use:
-----------
1. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
2. Type your blog title and content into the form.
3. Click "Save Entry" to store your post.
4. Scroll down to view previous entries.

Requirements:
-------------
- No installation required
- A web browser that supports HTML5 and JavaScript

Optional Improvements:

- Sync entries to a backend or database
- Add tags or search functionality
